# Style Modules

`styles.css` is the CSS entry file and imports these files in order. Keep the
order because later files intentionally override earlier shared styles.

- `00-base-home.css`: tokens, reset, shell, home page basics.
- `10-reports-services.css`: archive, report, upload, and service UI.
- `20-schedule-tasks.css`: schedule page, task panel, and shortcut cards.
- `30-metrics.css`: metric cards, metric detail pages, metric record dialogs.
- `40-medicine.css`: medicine check-in, records, detail, image preview.
- `50-diet-sport.css`: diet recognition/detail and sport UI.
- `60-checkin-forms.css`: bottom sheets, check-in forms, pickers, responsive rules.
